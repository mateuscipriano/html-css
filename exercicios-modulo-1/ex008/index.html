*Texto Deletado

é possível marcar um texto como deletado/excluído, indicando que deve ser lido mas não considerado, para isso, basta utilizar a tag "del"

*Texto Inserido/Sublinhado

para marcar um Texto Sublinhado, é possível utilizar a tag "ins" (Semântica)

para marcar um Texto Sublinhado também é possível utilizar a tag "u" (não semântica).

*Texto Sobrescrito

para inserir texto sobrescrito como: x20+5, é possível utilizar a tag "sup".

*Texto Subscrito

para inserir texto subscrito como: H2O, é possível utilizar a tag "sub".
